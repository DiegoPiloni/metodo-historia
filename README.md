# Método Historia - Sitio Web

Sitio web para clases particulares de historia online.

## Configuración

Para que el sitio funcione correctamente, necesitas agregar las imágenes en la carpeta `assets/images/`:

1. **Logo**: Guarda el logo como `assets/images/logo.png`
2. **Testimonios**: Guarda los screenshots de Instagram como:
   - `assets/images/testimonial-1.jpg` (primer testimonio - 4 estrellas)
   - `assets/images/testimonial-2.jpg` (segundo testimonio - 5 estrellas)
   - `assets/images/testimonial-3.jpg` (tercer testimonio - 4 estrellas)
   - `assets/images/testimonial-4.jpg` (cuarto testimonio - 5 estrellas)

## Uso

Simplemente abre el archivo `index.html` en tu navegador para ver el sitio.

## Características

- Diseño responsive (se adapta a móviles y tablets)
- Carousel automático de testimonios
- Botones de WhatsApp con mensaje pre-configurado
- Diseño con colores pastel y efecto papiro
- Todo en un solo archivo HTML

## Modificar el número de WhatsApp

Si necesitas cambiar el número de WhatsApp, busca en `index.html` el número `5493517158688` y reemplázalo por el nuevo número en formato internacional (sin espacios ni guiones).
