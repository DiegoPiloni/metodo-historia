# Método Historia - Sitio Web

Sitio web para clases particulares de historia - presenciales en Córdoba y online en toda Argentina.

## Configuración

Para que el sitio funcione correctamente, necesitas agregar las imágenes en la carpeta `assets/images/`:

### Imágenes requeridas

1. **Logo**: `assets/images/logo.png`
2. **Fondo del hero**: `assets/images/hero-bg.jpg`
3. **Foto de la profesora**: `assets/images/agus.png`
4. **Iconos de redes sociales**:
   - `assets/images/whatsapp.png`
   - `assets/images/instagram.png`
   - `assets/images/google-maps.png`
5. **Testimonios** (screenshots de Instagram):
   - `assets/images/testimonial-1.jpg`
   - `assets/images/testimonial-2.jpg`
   - `assets/images/testimonial-3.jpg`
   - `assets/images/testimonial-4.jpg`
   - `assets/images/testimonial-5.jpg`

## Uso

Simplemente abre el archivo `index.html` en tu navegador para ver el sitio, o visita [metodohistoria.com.ar](https://metodohistoria.com.ar).

## Características

- Diseño responsive (se adapta a móviles y tablets)
- Carousel infinito de testimonios con navegación manual
- Botones de WhatsApp con mensaje pre-configurado
- Botón flotante de WhatsApp (solo en móvil)
- Sección "Sobre la Profesora" con credenciales
- Links a Instagram y Google Maps en el footer
- Timeline decorativo en desktop
- Animaciones de entrada con Intersection Observer
- Diseño con colores papiro antiguo
- Todo en un solo archivo HTML

## Modificar el número de WhatsApp

Si necesitas cambiar el número de WhatsApp, busca en `index.html` el número `5493517158688` y reemplázalo por el nuevo número en formato internacional (sin espacios ni guiones).
