# Claude Instructions

## Versioning

Before committing changes, update the page version using semantic versioning (SemVer).

The version is located in `index.html` in the footer section:
```html
<p class="footer-version">vX.Y.Z</p>
```

### Semantic Versioning Rules

- **MAJOR (X)**: Increment for breaking changes or major redesigns
- **MINOR (Y)**: Increment for new features or significant enhancements
- **PATCH (Z)**: Increment for bug fixes, small improvements, or minor updates

### Examples

- Bug fix or typo correction: `v1.0.2` → `v1.0.3`
- New section or feature added: `v1.0.2` → `v1.1.0`
- Complete redesign: `v1.0.2` → `v2.0.0`
